# DOOMtang

This project aims to produce an implementation of DDOM in a Tang 20k Primer FPGA board, using a RISC-V softcore. To do so, a set of developement steps will be followed: 

1) Architecture simulation using spartan3E code.
2) DUT implementation according to Tang 20k Primer Dock, using RISC-V ALU.
3) C code devevolment and compilation to RISC-V assembly.
4) Board emulation and validation using C/C++ code.
5) DOOM compilation and execution in RISC-V softcore on Tang 20k Primer FPGA.
